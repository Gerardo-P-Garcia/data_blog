Following the outbreak of COVID-19, the Black Lives Matter movement brought attention to police violence against the Black community and the people overall, with protests spanning the globe bringing additional scrutiny and evidence against long-established practices. A wealth of information has followed, including self-recorded videos and data, allowing us to connect systemic-level violence with the intensity of individual shootings that some may have dismissed as anecdotal. # Functions for Visualizations

# Paste percent symbol
label_percent <- function(x){
  x <- as.character(x)
  for(i in 1:length(x)){
    if(i==1){
      x[i] <- paste(as.numeric(x[i])*100, "%", sep="")
    } else{ 
      x[i] <- paste(as.numeric(x[i])*100, "%", sep="")
    }
  }
  return(x)
}

label_bold <- function(x){
  x <- as.character(x)
  for(i in 1:length(x)){
    if(i==1){
      x[i] <- paste("<b>", x[i], "</b>", sep="")
    } else{ 
      x[i] <- paste("<b>", x[i], "</b>", sep="")
    }
  }
  return(x)
}

# Tables
create_dt <- function(x){
  require(DT)
  DT::datatable(x,
    extensions = 'Buttons',
    options = list(dom = 'Bfrtip',
      buttons = c('copy', 'csv', 'excel'),
      pageLength=10,
      scrollX=TRUE,
      lengthMenu = list(c(10,25,50,-1),
        c(10,25,50,"All"))),
    rownames = FALSE)
}

Just over 50% of victims were white, despite accounting for 75% of the US population. Meanwhile, 27% of victims were Black (13% of the population) and 18% were Hispanic (19%).

Using the 2019 US Census to estimate the population sizes for each race in the last 5 years, we can begin to understand the racial disparities despite the number of white victims. This assumes racial demographics have remained constant for the last 5 years in the US.
